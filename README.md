# Polymarket Rewards Scanner

Инструмент для поиска оптимальных рынков на Polymarket для получения вознаграждений за предоставление ликвидности.

## Что делает

- Загружает рынки из вкладки «Вознаграждения» через Polymarket API
- Применяет 6 фильтров: категория, minSize, награда ≥$30/день, спред <5¢, конкуренция, анализ стакана
- Показывает стакан заявок (Order Book) с маркером нашей ставки
- Даёт прямые ссылки на рынки для размещения лимитных заявок

## Деплой

Автоматически деплоится на Vercel при push в `main`.

## Переключение на реальный API

В `index.html` найди строку:

```js
const USE_MOCK = true;
```

Измени на `false` — приложение начнёт работать с живыми данными Polymarket.

## Стек

- React 18 (через CDN, без сборки)
- Polymarket Gamma API + CLOB API
- Чистый HTML/CSS/JS — никаких зависимостей для установки
